Rust WASM Пример

Демонстрация взаимодействия JavaScript с WebAssembly, скомпилированным из Rust

Как это работает:

  1. Введите текст в поле ввода
  2. Нажмите кнопку "Proceed"
  3. JavaScript передает данные в WASM модуль
  4. Rust функция обрабатывает данные
  5. Результат возвращается и отображается

Создано с использованием Rust, WebAssembly и современных веб-технологий
